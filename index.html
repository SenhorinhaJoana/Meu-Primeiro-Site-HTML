Estrutura Básica do HTML

Voltar

O HTML (HyperText Markup Language), a chamada linguagem de marcação, é fundamental para a web. Não é uma linguagem de programação, mas é geralmente utilizada com JavaScript (linguagem de programação) e com o CSS para acriação de aplicações WEB.Um documento HTML para ser considerado válido e em conformidade com as convenções precisa seguir alguns paramentros pre estabelecidos. Os elementos que compõem o documento HTML, chamados de Tags, servem para orientar o navegador qual é a estrutura que está sendo moldada, ou seja, os textos, as imagens, os videos, os parágrafos, os links, etc. É possível que qualquer pessoa com um conhecimento básico sobre o HTML possa criar e modificar sites sem a necessidade de pagar ou adquirir licenças. desta maneira essa especificidade contribuiu para uma maior democratização e a disseminação da web. O objetivo principal do HTML é posibilitar a elaboração de páginas da web com uma estrutura determinada e significativa, a partir da implementaçãoo de elementos hierarquizados de modo ordenado seguindo uma sequência lógica. O HTML é a base para a estruturação de páginas da web não dinâmicas, servindo como ponto inicial para a criação de sites dinâmicos e interativos (estilizados posteriormente com CSS e imbuídos de lógica com o JavaScript). É possível usar a linguagem na criação de e-mails e newsletters, pois oferece aos profissionais de marketing digital a possibilidade de criar layouts, adicionar imagens, estilizar texto e inserir chamadas para ação (CTAs) nas mensagens. A sintaxe do HTML foi criada para ser de fácil entendimento tanto para nós quanto para a máquina, além disso você não precisa de nada mais do que um editor simples. Como vimos, HTML é uma linguagem de marcação. As marcações do HTML são chamadas de tags, que são abertas e depois fechadas. É com as tags que dizemos para o navegador o que é um título, parágrafo, botão, formulário, dentre outros.

As principais Tags

Voltar

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML de todas as páginas da internet.

  1. html
  2. head
  3. title
  4. body
  5. Cabeçalho (h1, h2, h3, h4, h5, h6)
  6. Parágrafo (p) A tag (p) deve ser aberta e fechada para definir um parágrafo.

Atributos da Tags

Voltar

Os atributos são usados para personalizar as tags. Isso quer dizer que, de alguma forma, você pode desejar redimencionar o tamanho de uma imagem ou alterar a formatação de tabela ou mudar o tamanho e a cor da fonte. todas essas alternativas são possíveis com a ajuda de atibutos. A maioria das tags tem seus próprios atributos. Existem atributos que são especificos a determinadas tags como a tag IMG. Mas também existem atributos mais gerais que compreendem um grupo vasto de tags. Os Atributos são postos entre chevrons de uma tag aberta.

Listas Ordenadas e não Ordenadas

Voltar

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

Utilizando links

Voltar

No HTML, os links são definidos pela tag . Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link. Dentro do conteúdo da tag , incluímos então o texto ou elemento que servirá como redirecionador, ou seja, que ao ser clicado, executará a função de redirecionar para o endereço dentro do atributo href.
