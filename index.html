Estrutura Básica do HTML

Um documento HTML para ser considerado válido e em conformidade com as convenções precisa seguir alguns paramentros pre estabelecidos. Os elementos que compõem o documento HTML, chamados de Tags, servem para orientar o navegador qual é a estrutura que está sendo moldada, ou seja, os textos, as imagens, os videos, os parágrafos, os links, etc.
